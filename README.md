# 科學繪圖探究室

Scientific Illustration Lab 是一套以臺灣繁體中文製作的單頁式互動教材，帶領國小中高年級與國中學生從「仔細觀察」開始，學習忠實記錄比例、構造、紋理、比例尺與科學標示。

## 專案狀態

- 版本：v0.1.0（專案初始化）
- 形式：HTML5、CSS3、原生 JavaScript
- 執行：可直接開啟 `index.html`，不需要登入或後端
- 部署：預定 GitHub Pages

完整教材內容、使用方式、課綱對應、專案結構與公開網址將於正式版本補齊。
