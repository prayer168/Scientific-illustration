# Image 2.0 圖片產製與查證紀錄

共通風格：專業自然科學繪圖結合學生觀察筆記；細緻石墨線條、少量淡彩、乾淨米白紙張；教學寫實、構造正確、比例合理；不在圖內放文字、字母、數字或標籤；無浮水印；網頁橫向構圖。網站以 HTML 疊加標題與標示。

所有成品均使用內建 Image 2.0 產生並轉為 WebP。查證狀態於產圖檢視後更新；「通過」代表適合作為概念教學示意，不代表可取代物種鑑定。
