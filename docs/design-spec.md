# 視覺與互動設計規格

## 版面骨架

```text
┌──────────────────────────────────────────────┐
│ 品牌／進度／主題／全螢幕／選單              │
├──────────────────────────────────────────────┤
│ 12 單元頁籤（桌機橫向；手機抽屜）           │
├──────────────────────────────────────────────┤
│                                              │
│ 單一作用中的 tabpanel                       │
│ ┌───────────────┐ ┌──────────────────────┐ │
│ │ 圖像或互動證據 │ │ 短文、提示與即時回饋 │ │
│ └───────────────┘ └──────────────────────┘ │
│                                              │
├──────────────────────────────────────────────┤
│ 上一單元／單元計數／下一單元                 │
└──────────────────────────────────────────────┘
```

首頁主視覺也在第一個 `tabpanel` 內。每次只顯示一個主要學習單元，切換後將焦點送至單元標題；網址 hash 反映目前單元，可分享與使用瀏覽器返回。

## 色彩與材質

| Token | 淺色 | 深色 | 用途 |
| --- | --- | --- | --- |
| `--forest` | `#234c36` | `#9ed3ad` | 主標題、主要操作 |
| `--leaf` | `#4f7c55` | `#79b985` | 成功、進度 |
| `--paper` | `#fffdf5` | `#202620` | 卡片紙張 |
| `--cream` | `#f4edda` | `#161b17` | 頁面背景 |
| `--amber` | `#c86f35` | `#f0a56d` | 重點與警示 |
| `--sky` | `#4b88a8` | `#8ac8e5` | 資訊標籤 |
| `--graphite` | `#29332c` | `#eef3ed` | 主要文字 |
| `--specimen` | `#775642` | `#c39a7d` | 邊框與標本語彙 |

紙張效果只使用低對比漸層與細點，不壓低可讀性。文字與背景維持 WCAG AA；焦點框至少 3px，不能只靠顏色表達狀態。

## 字體與尺度

- 系統中文字型：`system-ui, "Noto Sans TC", "Microsoft JhengHei", sans-serif`。
- 正文 `clamp(1rem, .96rem + .2vw, 1.125rem)`，行高 1.75。
- 主要標題 `clamp(2.25rem, 6vw, 5.3rem)`；區段標題 `clamp(1.65rem, 3vw, 2.6rem)`。
- 每行正文約 38–70 個中文字；卡片至少 16px 內距。

## 元件

- `field-note`：米白紙卡、左上短標籤、手繪虛線。
- `evidence-card`：圖像、觀察證據、提醒三層。
- `tool-card`：工具圖示、用途、階段、提醒、替代工具。
- `myth-card`：翻面卡；按鈕語意、`aria-expanded`、鍵盤可操作。
- `hotspot`：44×44px，數字與名稱同時顯示，選取後有文字回饋。
- `choice-chip`：多選狀態使用勾號、文字與色彩三重提示。
- `feedback`：`role="status"`；答對、待修正、提示不只用圖示。
- `modal`：作品放大檢視；焦點限制、Escape 關閉、關閉後回原按鈕。

## 斷點

- `>= 1100px`：內容雙欄、12 頁籤可橫向瀏覽。
- `720–1099px`：內容單欄或 2 張卡片並排。
- `< 720px`：行動版選單；所有互動改單欄；拖曳活動同時顯示點選替代介面。
- 全寬元件設定 `min-width: 0`、圖片 `max-width: 100%`；整頁禁止水平溢出。

## 動畫與無障礙

- 單元切換 180ms 淡入，不使用閃爍。
- 葉片八步驟以狀態顯示 SVG/HTML 疊加標示；播放、暫停、重播、前後步驟與速度選擇同步更新文字。
- `prefers-reduced-motion: reduce` 時停用平移、縮放與自動播放。
- 所有操作提供可見標籤、鍵盤事件、觸控尺寸、即時文字回饋。
- 圖片若載入失敗，保留 `alt` 並顯示帶檔名的替代卡。

## 列印

- 隱藏導覽、按鈕、圖庫與非紀錄單元。
- 只列印觀察紀錄、自我檢核與基本欄位；A4 直式、12mm 邊界。
- 空白學習單獨立為 `downloads/scientific-drawing-worksheet.html`，可直接開啟與列印。
